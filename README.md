# The Work Culture тАУ Frontend Website Redesign

## Project Overview

This project is a **frontend website redesign** developed as part of a technical assignment provided by **Kiwi Digital, Bhopal**.

The objective was to redesign the existing **The Work Culture** website with a modern, responsive, and user-friendly interface while preserving the original business information, services, and branding.

The redesign focuses on improving the overall user experience, visual hierarchy, responsiveness, accessibility, and modern UI standards.

## Installation

Clone the repository

```bash
git clone <repository-url>
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
